# woowa-open-mission

# 자동차 경주 게임을 진행한 뒤, 경주 결과(1~3등)에 따라 로또를 진행할 수 있는 프로그램

- 자동차 레이싱에서 우승한 1,2,3 등에게만 로또를 할 수 있는 기회가 주어진다.
- 1등은 10번, 2등은 5번, 3등은 3번의 로또를 할 수 있는 기회가 주어진다.

## 배포 주소  (배포 중단)

- http://ec2-54-165-56-164.compute-1.amazonaws.com:8080/lottoRacing/

## 기술 스택

- 언어: Java 21
- IDE: Eclipse
- 배포/운영: AWS EC2, Docker
- 버전 관리: Git, GitHub
- CI/CD: GitHub Actions (자동 빌드, 테스트, 배포)

## 기능요구 사항

### 1. 게임 머니 관리

1-1) 게임 머니 금액 입력 기능

- 최소 5000원 이상, 최대 100000 이하 입력해야 한다.

### 2. 자동차 경주

2-1) 경주할 자동차 이름 입력 기능

- 자동차 경주 시작 금액 입력 에서 1000으로 나눈 몫의 갯수대로 자동차 입력을 해야 한다.
  (ex 5000원 입력시 5대의 자동차 입력, 10000원 입력시 10대의 자동차 입력)
- 각 자동차의 이름의 길이는 1 ~ 5자 이하여야 하며 공백은 허용되지 않는다.
- 각 자동차의 구분은 쉼표(,) 로 한다.

2-2) 1~3등 순위권 자동차 이름 입력 기능

- 경주할 자동차 이름 입력한 것 중에서 사용해야 한다.
- 3개를 입력해야 한다.

2-3) 자동차 운행 기능

- 자동차는 전진, 후퇴, 멈춤을 한다.
- 랜덤값으로 전진, 후회, 멈춤을 판단하는데 Console API의 Random.pickNumberInRange() 를 사용하여 랜덤값을 얻는다.
- 0은 멈춤, 1~4는 후퇴, 5~9는 전진이다.

2-4) 우승자 계산 기능

- 전진은 +1, 후퇴는 -1, 멈춤은 0으로 계산하고 합산을 하여 우승자를 결정한다.
- 합산한 점수가 동점일 경우 전진 수가 많은 순서대로 결정한다.
- 합산한 점수, 전진수가 동점일 경우는 후진 수가 적은 순서대로 결정한다.
- 합산한 점수, 전진 수, 후진 수가 동점일 경우는 멈춤 수가 많은 순서대로 결정한다.
- 위의 모든 사항이 같을 경우는 이름 오츰차순으로 한다.

### 3. 로또 기능

3-1) 로또 시도 기능

- 자동차 레이싱에서 1,2,3등에게만 기회가 주어진다. 1등은 10번, 2등은 5번, 3등은 3번의 기회가 주어진다.

3-2) 당첨 번호 입력 기능

- 1 ~ 45 사이의 숫자를 입력해야 한다.

3-3) 보너스 번호 입력 기능

- 1 ~ 45 사이의 숫자를 입력해야 한다.
- 당첨 번호 입력에서 입력하지 않았던 번호를 입력해야 한다.

3-4) 로또 당첨 기능

- 1, 2, 3 등에게 주어진 횟수만큼 로또 당첨 번호를 출력한다. MissionUtils.Random.pickUniqueNumbersInRange를 사용하여 랜덤 당첨번호를 낸다.
- 입력한 당첨번호와 각 자동차들의 랜덤 당첨번호를 비교하여 계산한다.
- 당첨이 되는 경우의 수는 아래와 같다.

| 3개 맞은경우               | 5등 |     |
| -------------------------- | --- | --- |
| 4개 맞은경우               | 4등 |     |
| 5개 맞은경우               | 3등 |     |
| 5개 맞은경우 + 보너스 번호 | 2등 |     |
| 6개 맞은경우               | 1등 |     |

## 예외 사항

### 1. 게임 머니 관리

- 빈문자열을 입력한 경우
- 숫자가 아닌 다른 타입을 입력한 경우
- 5000원 미만 입력시 예외 발생
- 1000원으로 나누어 떨어지지 않는 경우

### 2. 자동자 경주

2-1) 경주할 자동차 이름 입력 기능

- 자동차 이름 갯수가 2~100 사이가 아닌경우
- 자동차 이름 길이가 1~5 사이가 아닌 경우
- 자동차 이름에 공백이 있는 경우 (ex: y una)
- 자동차 이름이 중복된 경우
- 자동차 구분을 쉼표(,)가 아닌 다른 문자로 구분한 경우

2-2) 1~3등 순위권 자동차 이름 입력 기능

- 경주할 자동차에 입력한 자동차 외에 다른 이름을 쓴 경우
- 자동차 이름 3개를 쓰지 않은 경우

### 3. 로또

3-1) 당첨 번호 입력 예외 처리

- 빈문자열을 입력한 경우
- 쉼표 구분자(,)가 아닌 다른 구분자로 번호를 구분한 경우
- 입력한 값이 숫자가 아닌 다른 타입의 값이 있을 경우
- 6개의 숫자를 입력하지 않았을 경우
- 입력한 숫자에서 중복된 숫자가 있는 경우
- 입력한 숫자가 1~45 범위를 벗어난 경우

3-2) 보너스 번호 입력 예외 처리

- 빈문자열을 입력한 경우
- 숫자가 아닌 다른 타입을 입력한 경우
- 입력한 숫자가 1~45 범위를 벗어난 경우
- 당첨 번호와 중복된 경우
